## Initial Setup

- **Default Search:** Use Brave Search or internal search. Remove Google.
- **Shields:** Set to Aggressive blocking at `brave://settings/shields`.
- **Disable:** Brave Rewards, Brave News, sponsored new tab images.
- **Sync:** Use Brave Sync chains — no Google account required.
- **Extensions:** Full Chrome Web Store compatibility. Install Bitwarden first.
- **Privacy:** Block 3rd-party cookies, disable feature-improvement telemetry.
- **IT Admins:** Group Policy (Windows) + MDM (macOS) supported for fleet deploy.

## Bookmarks

- **Structure:** Organize by dept/workflow, not recency. Keep toolbar to 8–10 items.
- **Team Sharing:** Export canonical `bookmarks.html` and let staff import — no sync needed.
- **Speed Dial:** Pin top 6 tools to the new tab page for one-click access.
- **Shortcuts:** Assign keyword shortcuts (e.g., type `crm` + Tab to jump to Salesforce).

## Tab Groups

- **Create:** Right-click any tab → Add to new group. Name it, pick a color.
- **Color Code:** Red = urgent, Blue = research, Green = client, Gray = reference.
- **Collapse:** Click group label to park all tabs — zero cognitive overhead.
- **Save:** Right-click group → Save group. Opens as a folder from bookmarks.
- **Workspaces:** Separate window contexts per client/project — no tab bleed.

## Bitwarden: Passwords, Secrets & Passkeys

### Password Manager

Browser extension auto-fills credentials, generates strong passwords, and syncs securely across your team via shared org vaults with role-based access.

### Secrets Manager

For DevOps: store API keys, tokens, and infrastructure secrets separately from user credentials, with audit logs and fine-grained access controls.

### Passkeys

Full passkey support — phishing-resistant, hardware-backed auth stored in your vault. Not locked to any one device. Works cross-platform across your whole team.

## Security Comparison

| Feature | Brave | Chrome |
|---|---|---|
| Tracker blocking | Built-in, aggressive default | Requires extensions |
| Fingerprinting | Blocked by default | Not included |
| Google telemetry | None sent | Significant data collection |
| Native ad blocking | Yes — no extension needed | No — MV3 limits this |
| Chrome extensions | Full compatibility | Full compatibility |
| Enterprise MDM | Yes (functional) | Yes (more mature) |
| Google Workspace | Full SaaS compatibility | Full + OS integration |
| Sync w/o Google | Yes — Brave Sync | Requires Google account |

**What Brave Can Replace**

Chrome as your everyday browser on Windows/macOS. All SaaS tools, web apps, extensions, SSO flows, and video conferencing work identically. Most users won't notice a difference.

**What It Cannot Replace**

Google Workspace itself (Docs, Sheets, Gmail) — those are SaaS tools. Chrome Device Management, ChromeOS fleet management, and deep Google MDM integrations remain Chrome's domain.

OpenTechnologyApp is a modern task queue manager built for teams who need real power without enterprise complexity. Manage customers, tasks, and items — with a client portal included — all under one roof.

→ [Visit the OpenTechnologyApp landing page](/open-technology-app)

## Features

### Task queues

Structured queues organize every item with status, priority, type, and assignee — all in one clean view.

<img src="/ota-queues.png" alt="Task queue view" style={{width: '100%', borderRadius: '12px', margin: '1.5rem 0'}} />

- Filter, sort, and export items at any time
- Parent/child item relationships for complex work
- Customizable columns per queue

### Live dashboards

Widget-based dashboards give teams a real-time view of workload, priorities, and progress.

<img src="/ota-dashboards.png" alt="Live dashboard with calendar and priority widgets" style={{width: '100%', borderRadius: '12px', margin: '1.5rem 0'}} />

- Calendar view with inline item previews
- Priority breakdown charts (Low / Medium / High)
- Queue distribution widgets with percentage splits

### Workflow automation

Trigger-based automations eliminate repetitive manual steps across every queue.

<img src="/ota-automations.png" alt="Workflow automations list" style={{width: '100%', borderRadius: '12px', margin: '1.5rem 0'}} />

- Auto-assign, escalate, and move items on condition
- Status-change triggers with comment injection
- Slack and email notification actions built in

### Client portal

An external-facing portal lets clients submit, track, and view their own items without shared logins.

<img src="/ota-portal.png" alt="Client portal view" style={{width: '100%', borderRadius: '12px', margin: '1.5rem 0'}} />

- Branded portal per queue with custom notices
- Clients see only their own submitted items
- Form fields fully configurable per portal

### By project

Projects group related queues, dashboards, portals, and automations into one organized space.

<img src="/ota-projects.png" alt="Project overview screen" style={{width: '100%', borderRadius: '12px', margin: '1.5rem 0'}} />

- Role-based access: Admin, Member, Viewer
- Multiple projects side by side in the sidebar
- Each project is fully self-contained


## Access Control

Role-based access enforced server-side. A demoted user's next request is rejected immediately — no restart, no delay.

<img src="/ota-roles.svg" alt="Open Technology App Role-Based Access Control" style={{width: '100%', borderRadius: '12px', margin: '1.5rem 0'}} />

- **Admin** — Manage users, all settings, automations, full data access, read/write items, comments, time logging
- **Member** — Read/write items, comments, time logging, full project access
  - Member role permissions are configurable — Member Roles coming soon
- **Viewer** — Read all items, time logging, public comments
